# Clear and Exit

## Clear

Clears the terminal.

#### Usage

```
clear
```

## Exit

Exits the CLI.

#### Usage

```
exit
```

#### Exit or Quit

exit and quit can be used to abort the current command.&#x20;

{% hint style="warning" %}
Any request inputs can be exited using one of the following inputs: .exit, .quit, .e, .q, exit
{% endhint %}

#### Example

```
 $main@uni-5> ado create cw721
- Fetching schema...
Input Minter: exit
Prompt exited
Use the --help flag for help
? $main@uni-5> () 
```

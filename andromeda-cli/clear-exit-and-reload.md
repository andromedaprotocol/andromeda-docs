# Clear, Exit and Reload

## Clear

Clears the terminal.

#### Usage

```
clear
```

## Exit

Exits the CLI.

#### Usage

```
exit
```

## Reload

Prints the previous commands.

#### Usage

```
relaod
```

#### Example

```
? $test@uni-3> reload
? Select from recent commands (or Close to exit)
 (Use arrow keys)
```
